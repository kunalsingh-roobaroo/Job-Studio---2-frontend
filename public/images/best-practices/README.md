## Image Specifications
- **Recommended size:** 800x400 pixels (2:1 aspect ratio)
- **Format:** JPG or PNG
- **File size:** Under 200KB for fast loading

## Tips
1. Use screenshots from LinkedIn (with permission or from your own profile)
2. Blur out personal information if needed
3. Add annotations/highlights to point out best practices
4. Keep images clean and professional
